# x402Resolve Multi-Oracle Demo

**Live Demo**: https://tanaka-kamiyo.github.io/kamiyo-mcp/

Multi-oracle consensus for API dispute resolution on Solana.

## Features

- **5 Independent Oracles** with different methodologies
- **Ed25519 Signature Verification** 
- **Median Consensus Algorithm** (resistant to outliers)
- **Quality-Based Refunds** (0-100% based on assessment)
- **SPL Token Support** (SOL, USDC, USDT)

## Network

- **Network**: Solana Devnet

## How to Use

1. Install Phantom Wallet
2. Switch to Devnet
3. Get devnet SOL from https://faucet.solana.com/
4. Visit demo and connect wallet
5. Test multi-oracle dispute resolution
